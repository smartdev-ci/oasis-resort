# COPILOTE ÉTABLISSEMENT IA — VISION PRODUIT

## Contexte

Tu es un Product Designer, Product Manager, UX Expert et Architecte SaaS spécialisé dans les plateformes de réservation d'hôtels, résidences meublées, maisons d'hôtes et restaurants en Afrique.

Ta mission est de concevoir l'expérience complète du module "Copilote Établissement".

L'objectif principal est :

> Permettre à un gérant d'établissement de gérer son activité en moins de 5 minutes par jour depuis son téléphone.

La solution doit être adaptée au contexte africain où de nombreux établissements utilisent encore WhatsApp, Excel ou des cahiers papier.

---

# Philosophie Produit

Les PMS classiques sont complexes.

Notre plateforme doit être :

- Mobile First
- IA First
- WhatsApp First
- Sans formation
- Accessible aux non-techniciens

L'utilisateur ne doit pas avoir l'impression d'utiliser un logiciel.

Il doit avoir l'impression de dialoguer avec un assistant personnel.

---

# Vision

Le tableau de bord traditionnel est remplacé par un Copilote IA.

L'établissement peut :

- créer des chambres ;
- modifier des tarifs ;
- consulter ses revenus ;
- gérer les réservations ;
- recevoir des recommandations ;
- administrer son personnel ;

simplement en discutant avec l'IA.

---

# Cycle de vie de l'établissement

## 1. Créé par la plateforme

L'établissement peut être créé automatiquement par notre équipe.

Sources possibles :

- Google Maps
- Réseaux sociaux
- Référencement local
- Partenaires terrain

Statut :

"Non revendiqué"

---

## 2. Revendiqué

Le propriétaire confirme qu'il est gestionnaire.

Méthodes :

- OTP SMS
- OTP WhatsApp
- Email
- Vérification documentaire

Statut :

"Revendiqué"

---

## 3. Configuré

Le propriétaire complète :

- photos ;
- chambres ;
- tarifs ;
- coordonnées.

Statut :

"Actif"

---

## 4. Optimisé

Le Copilote IA analyse :

- revenus ;
- taux d'occupation ;
- performance ;
- satisfaction client.

Statut :

"Optimisé"

# Assistant IA principal

L'assistant doit comprendre le langage naturel.

Exemples :

"Ajoute une nouvelle chambre"

"Augmente les prix de 10 %"

"Montre-moi les réservations de demain"

"Bloque la chambre 204"

"Quels sont mes revenus du mois ?"

"Pourquoi mes ventes baissent ?"

"Prépare une promotion pour ce week-end"

Chaque demande doit déclencher une action réelle.

---

# Création simplifiée des chambres

Le propriétaire peut écrire :

"J'ai 10 chambres standard et 3 suites."

L'IA génère automatiquement :

- catégories ;
- descriptions ;
- équipements ;
- capacité ;
- tarifs de base ;
- disponibilité.

---

# Gestion vocale

Le système doit accepter :

- notes vocales ;
- commandes vocales ;
- messages audio WhatsApp.

Exemple :

"Bloque la chambre 203 pour maintenance jusqu'à dimanche."

L'IA doit :

- transcrire ;
- comprendre ;
- exécuter l'action.

---

# Calendrier intelligent

Créer un calendrier ultra visuel.

Codes :

🟩 Libre

🟨 Arrivée prévue

🟥 Occupé

⬛ Maintenance

L'utilisateur doit comprendre le calendrier sans formation.

---

# Gestion WhatsApp

Toutes les actions importantes doivent pouvoir être réalisées depuis WhatsApp.

Exemple :

Nouvelle réservation

Client : Jean Dupont

Arrivée : 15 août

Départ : 18 août

Montant : 120 000 FCFA

Actions :

✅ Accepter

❌ Refuser

📞 Contacter

# Vérification automatique des clients

L'IA doit être capable :

- de scanner une pièce d'identité ;
- d'extraire les informations ;
- de remplir automatiquement la fiche client.

Informations extraites :

- nom ;
- prénom ;
- nationalité ;
- numéro de document ;
- date d'expiration.

- chambre ;
- salle de bain ;
- réception ;
- restaurant ;
- piscine ;
- terrasse ;
- façade.

Permettre :

- photos 360° ;
- vidéos ;
- visites guidées.

Fonctionnalités :

- changement de pièce ;
- navigation immersive.

# Gestion hors ligne

Le système doit fonctionner sans Internet.

Fonctionnalités :

- consultation locale ;
- modifications locales ;
- synchronisation automatique au retour du réseau.

Très important pour les zones à faible connectivité.

Canaux :

# Positionnement stratégique

Nous ne construisons pas :

- un PMS ;
- un dashboard ;
- un back-office.

Nous construisons :

# Le premier Copilote IA pour établissements africains

Capable d'agir comme :

- réceptionniste ;
- revenue manager ;
- assistant administratif ;
- support client ;
- analyste commercial.

L'objectif final est que le propriétaire dialogue avec son établissement comme il dialogue avec un employé.
